How to Apply for a Service
  1. Register and Login to the portal.
  2. Navigate to 'Apply for Services'.
  3. Select the desired department and service.
  4. Fill in the required details and upload documents.
  5. Submit the application and note the Application ID.
For Officers
Processing Applications
  1. Login with Officer credentials.
  2. View 'Pending Applications' in dashboard.
  3. Verify documents and add remarks.
  4. Approve or Reject the application.
